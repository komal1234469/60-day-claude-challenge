# Day 48 – The Verdict Engine ⚖️

## AI-Powered Decision Comparison Tool

**Day 48 of my 60 Days Coding Challenge**

The Verdict Engine is an interactive decision-support application designed to help users compare multiple options using measurable criteria, weighted priorities, and real research-backed data.

The application transforms complex comparisons into a transparent and easy-to-understand ranking system.

---

## 🎯 Objective

The goal of this project was to build a decision-making tool that allows users to:

* Compare different options
* Evaluate options using multiple criteria
* Assign custom weights based on personal priorities
* View rankings that update dynamically
* Understand how the final verdict was calculated
* Verify the data through cited sources
* Review the research methodology and source conflicts

---

## ✨ Key Features

### 1. Interactive Comparison

Users can compare multiple options against measurable criteria such as:

* Cost
* Time
* Risk
* Quality
* Availability

### 2. Custom Criteria Weighting

Users can adjust the importance of each criterion according to their personal priorities.

The ranking updates live whenever the weights are changed.

### 3. Dynamic Ranking

The application calculates weighted scores and presents a ranked result based on the selected criteria priorities.

### 4. Transparent Sources

A dedicated sources panel displays the sources used for the research and data points.

### 5. Research Methodology

The collapsible **"How this was researched"** section explains:

* Where the data came from
* How data points were verified
* Which sources were used
* Any conflicts found between sources
* How conflicting information was resolved

### 6. Data Transparency

The application clearly identifies estimates or synthetic placeholders instead of presenting them as sourced facts.

### 7. Responsive Design

The application is designed to work across:

* Desktop
* Tablet
* Mobile

## 🔄 How It Works

1. Select the options to compare.
2. Define measurable comparison criteria.
3. Review the researched data.
4. Adjust the weight of each criterion.
5. Observe the ranking update live.
6. Review the sources panel.
7. Open the research methodology section.
8. Understand conflicts and how they were resolved.
9. Make a more informed final decision.
