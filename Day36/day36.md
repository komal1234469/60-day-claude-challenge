# Day 36 – Cognitive Pattern Explorer

## Overview

Today I built **Cognitive Pattern Explorer**, a psychology-inspired self-reflection web application designed to help users explore their thinking tendencies through interactive scenarios and activities.

The application is designed as an educational and reflective experience rather than an assessment tool. It encourages users to observe how they approach decisions, uncertainty, priorities, and actions in different situations.

## Features

* Calm Mode and Stress Mode experiences
* Interactive scenario-based thinking exploration
* Drag-and-drop priority ranking activity
* Thinking timeline arrangement exercise
* Personalized Reflection Journal
* Percentage-based thinking tendency breakdown
* Responsive design for desktop and mobile devices
* Fully offline functionality
* Built using only HTML, CSS, and Vanilla JavaScript

## Thinking Tendencies Explored

* Analytical Thinker
* Emotional Intuitive
* Overthinking Loop Style
* Action-First Decision Maker
* Balanced Reflective Thinker

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Native Drag and Drop API

## Key Learnings

* Designing engaging self-reflection experiences without making clinical assessments.
* Implementing reusable drag-and-drop components using native browser APIs.
* Creating smooth user flows with progress tracking and transitions.
* Building a complete offline-first application using only frontend technologies.
* Improving responsive design and accessibility considerations.
