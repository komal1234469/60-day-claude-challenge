# 🧩 Prompt Puzzle — Master AI Prompting Through Play

## Day 35 — Interactive Prompt Engineering Challenge

Built as part of the **60 Days AI Challenge by ABTalksOnAI**.

## 🚀 Project Overview

Prompt Puzzle is an interactive learning game designed to help users improve their prompt engineering skills through hands-on practice.

The application presents users with realistic AI scenarios and challenges them to construct, optimize, and evaluate prompts using core prompt engineering principles.

The entire application runs completely offline inside a single HTML file with no external dependencies.

---

## ✨ Features

* 🎯 6–8 randomized prompt engineering scenarios
* 🧠 Practice across multiple domains and difficulty levels
* 🧩 Three challenge types:

  * Build the Prompt
  * Clean the Prompt
  * Choose the Best Prompt
* 📊 Live scoring system using:

  * Accuracy
  * Time
  * Moves
  * Wrong Placements
  * Hints Used
  * Optimization Bonus
* 📈 Prompt Performance Report including:

  * Prompt Score
  * Rating
  * Rank
  * Prompt DNA Visualization
  * Personalized Feedback
  * Next Milestone
  * Final Optimized Prompt
* 🔄 Replay mode with new randomized scenarios
* 🎨 Premium UI with animations, drag-and-drop interactions, and progress indicators
* 💾 Fully offline functionality

## 🎯 Key Learnings

* Small prompt improvements can significantly enhance AI output quality.
* Clear instructions outperform vague requests.
* Context and constraints are essential components of effective prompts.
* Over-engineering prompts can sometimes reduce performance.
* Interactive learning improves prompt engineering retention.
