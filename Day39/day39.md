# Day 39 - PDF Splitter & Merger

## Create a Premium Browser-Based PDF Utility

PDF Splitter & Merger is a browser-based document processing application designed to provide professional-grade PDF manipulation directly within the browser.

## Objective

The goal of this project was to create a fast, secure, and user-friendly PDF utility capable of handling common document workflows without requiring server-side processing.

## Features

### PDF Splitter

Users can:

* Detect total pages automatically
* Preview uploaded documents
* Split using custom page ranges
* Split after selected pages
* Split every N pages
* Extract selected pages into new files
* Validate page ranges before processing

### PDF Merger

Users can:

* Upload multiple PDF files
* Reorder files using drag-and-drop
* Preview uploaded documents
* View total page count
* Generate merged output files
* Download processed PDFs

### User Experience Features

* Drag-and-drop uploads
* Loading indicators
* Smooth animations
* Responsive design
* Accessibility support
* Keyboard shortcuts
* Error handling and validation
